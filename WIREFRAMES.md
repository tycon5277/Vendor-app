# Component Specifications

## Common UI Patterns

### Card Component
```jsx
// Standard card with shadow
<View style={{
  backgroundColor: '#FFFFFF',
  borderRadius: 20,
  padding: 16,
  shadowColor: '#6366F1',
  shadowOffset: { width: 0, height: 4 },
  shadowOpacity: 0.1,
  shadowRadius: 12,
  elevation: 4,
}}>
```

### Primary Button
```jsx
<TouchableOpacity style={{
  backgroundColor: '#6366F1',
  borderRadius: 16,
  paddingVertical: 16,
  paddingHorizontal: 24,
  alignItems: 'center',
}}>
  <Text style={{ color: '#FFFFFF', fontWeight: '700', fontSize: 16 }}>
    Button Text
  </Text>
</TouchableOpacity>
```

### Tab Selector (Pill Style)
```jsx
// Used in Discounts list (Active/Scheduled/Past)
<View style={{ 
  flexDirection: 'row', 
  backgroundColor: '#F3F4F6', 
  borderRadius: 12, 
  padding: 4 
}}>
  <TouchableOpacity style={{
    flex: 1,
    paddingVertical: 10,
    borderRadius: 10,
    backgroundColor: isActive ? '#6366F1' : 'transparent',
  }}>
    <Text style={{ color: isActive ? '#FFFFFF' : '#6B7280' }}>Tab</Text>
  </TouchableOpacity>
</View>
```

# API Endpoints Needed

## Discounts
- `POST /api/vendor/discounts` - Create discount
- `GET /api/vendor/discounts` - List discounts
- `PUT /api/vendor/discounts/{id}` - Update discount
- `DELETE /api/vendor/discounts/{id}` - Delete discount
- `GET /api/vendor/discounts/{id}/analytics` - Discount performance

## Timings
- `GET /api/vendor/timings` - Get operating hours
- `PUT /api/vendor/timings` - Update operating hours
- `POST /api/vendor/timings/holidays` - Add holiday
- `DELETE /api/vendor/timings/holidays/{id}` - Remove holiday
- `POST /api/vendor/timings/close-early` - Close shop early today

## Shop QR
- `GET /api/vendor/qr/shop` - Get shop QR data
- `GET /api/vendor/qr/products` - Get product QR list
- `PUT /api/vendor/qr/customize` - Save QR customization
- `GET /api/vendor/qr/analytics` - QR scan stats

## Promote
- `GET /api/vendor/promotions` - List promotions
- `POST /api/vendor/promotions/featured` - Create featured listing
- `POST /api/vendor/promotions/boost` - Create visibility boost
- `GET /api/vendor/promotions/analytics` - Campaign performance
- `POST /api/vendor/share/whatsapp` - Generate share content
